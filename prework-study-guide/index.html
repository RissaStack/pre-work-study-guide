HTML

  • HTML is used for the foundation on a website.
  • HTML is made up of elements to create this foundation.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head and body elements are going to be the parent elements that hold the child elements inside them.
  • Use h1, h2, h3 and so on the make new headers. Only use h1 for the top title.
  • Use the p element to seperate paragraphs.
  • Use the img element to insert an image.
  • The src element is used for the location of the image.
  • The br element is used to make a break of empty space on the webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we wnat around the content inside an element.
  • Use * when you want a rule to apply to all elements on the page.
  • Use commas when you want to combine two selectors into a single CSS rule. DRY rule!
  • You must link your style sheet to your HTML file for it to show up.
  • You can also create style without using an external styling sheet by using class attributes.
  • When you add a .card you are giving all the elements the same rule.
  • You can add color using the HTML color code.

GIT

  • Git is a free source to collaborate and save projects.
  • Never work on the main branch in Git. Always create a new branch so you do not make mistakes on the main branch.
  • Have other team members chack your work before adding your work to the main branch.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • JavaScript is what makes a webpage interactiev.
  • Variable is a named container that allows us to store data in our code. The variable name must be unique.
  • To assign data to the variable, you must use an =.
  • Each line of JavaScript code ends in ;
  • Chrome DevTools is used to view console.log output.
  • Control Flow is the order in which a computer executes code in a script.
  • We can change the flow using conditional statements: if.
  • An array is a group of data in a single ariable.
  • An index is a location number. Every index begins with 0 and goes up by 1 each time.
  • A for loop uses predictable patterns of indecies to perform a task on all items in an array by allowing a single code block to be executed over and over.
  • Commenting code makes code inactive without deleting it. You do this by using /* at the beginning of the code and ending with */.
  • A function is a set of instructions that tells the computer how to perform a task.
  • Functions do NOT run automatically like a loop or conditional statment. YOu must call for the function by name to execute it.
  • A function name is always followed by ()
